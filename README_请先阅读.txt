🔍 修复了什么问题？

简单来说：

❌ 修复前：
   你的 Cursor 登录凭证（Token）会被发送到一个
   第三方服务器（token.cursorpro.com.cn）

   就像把你家钥匙交给陌生人保管 - 很危险！

✅ 修复后：
   所有处理都在你自己的电脑上完成
   不会发送任何数据到外部

   钥匙一直在你自己手里 - 安全！
